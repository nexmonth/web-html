HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. Dengan HTML, kita dapat menentukan elemen-elemen di sebuah halaman web seperti:

  • Heading (judul halaman)
  • Paragraph (teks paragraf)
  • Image (gambar)
  • Link (tautan ke halaman lain)
  • Table (tabel)
  • Form (formulir untuk input data)
